三
功能详细说明

3.1 布置练习教辅

教师在主页点击「练习教辅」任务类型，进入发布页面，完成配置后发布。

功能入口主页「布置任务」卡片 → 点击「练习教辅📷」
页面选择左侧课本目录展示章节结构，右侧逐页选取，支持多页勾选
截止时间必填，默认为今天 23:59
发布确认点击底部「确认布置」，弹出发布成功弹窗
通知方式发布成功弹窗提供小程序二维码 + 可复制文字通知，便于发至家长群
多页选择时，系统将选中页面合并为一次作业任务，学生需逐页拍照提交所有选中页面。

3.2 任务详情

教师点击主页「最近任务」或「全部任务」列表中的任务记录，进入任务详情页。

任务头部任务标题、所属书本与班级、发布时间、截止时间
作业内容可折叠展示，列出本次作业涉及的书本页面
完成概况提交情况圆环（已提交/共 N 人）+ 平均错误率圆环
学生详情见 3.3

3.3 学生作答详情

在任务详情页底部，以表格形式呈现全班学生的提交与作答情况。

筛选标签已提交 (N) / 未提交 (N)，默认展示已提交
已提交行学生姓名、错误率、提交时间、查看按钮
未提交行学生姓名、「—」占位
查看详情点击弹出右侧抽屉，展示该学生全部拍照及逐题批改结果

学生详情抽屉内容：

区域内容
头部学生姓名、提交时间、错误率
拍照预览逐页显示拍照图片，点击可放大
逐题批改题目名称、题型、对错标记（✓ / ✗）
错误率数据由学生提交拍照后系统自动识别批改，教师无需手动录入。
四
界面布局说明
主页布局左侧班级信息栏 / 中间内容区（书本、布置任务、最近任务）/ 右侧资源区
发布页布局左侧课本目录 TOC / 右侧配置区（页面选择 + 截止时间），底部确认按钮
任务详情布局左侧任务记录列表 / 右侧详情滚动区（头部 + 内容 + 概况 + 学生表格）
学生详情抽屉从右侧滑入，覆盖详情区右半部分，点击蒙层关闭
发布成功弹窗居中弹窗，左栏二维码 + 右栏文字通知，仅 ✕ 关闭
五
数据与权限说明
课本目录由编辑预先配置，与现有书本数据同源
作业任务教师发布后生成，包含选页、截止时间、班级信息
提交数据学生从微信小程序扫码上传，服务端接收拍照后自动批改
错误率统计系统自动计算，按题目粒度展示，不需要教师手动操作
可见范围教师只能看到自己班级学生的提交数据
